# BOLVA CEO 經營決策控制艙 (2025) 指引

本技能旨在管理與維護一套專為 CEO 打造的高級經營看板。該看板基於 Streamlit + Plotly 構建，核心目標是將雜亂的 Excel 財務數據轉化為具備「戰略指導意義」的動態決策引擎。

## 🚀 核心能力
1. **多維財務度量**：自動解析「年度利潤」、「銷售數據」、「銀行餘額」、「平台費用」與「運營費用」。
2. **2026 營收預測**：支持「悲觀/穩健/進取」三種場景的動態模擬，並與當前季度篩選聯動。
3. **自動化 AI 洞察**：為每個圖表生產 20 字以內的精煉結論與詳細決策建議（Popover 形式）。
4. **深度毛利穿透**：提供客戶與業務員的毛利分析，包括 Pareto 貢獻曲線、效率矩陣。
5. **數據鮮度控制**：基於文件指紋（MTime + Size）的自動緩存失效機制與一鍵強制刷新。

## 📋 數據與環境要求

### 1. Excel 數據結構 (重要)
看板依賴 `2025年全年.xlsx` 中的以下 Sheet：
- **年度利潤**：需包含「月份」、「銷售額」、「淨利潤」。
- **銷售數據**：關鍵列需包含「日期」、「購貨單位」、「產品名稱」、「銷售收入」、「銷售成本」、「銷售毛利」、「業務員」。
- **平台 銷售費用比**：記錄各平台（Amazon, TikTok 等）的佣金、配送、廣告率。
- **銀行餘額**：需有名為「銀行餘額」的 Sheet 且包含「本位幣(CNY)」列。
- **運營費用**：自動查找包含「日期」與「金額」表頭的 Sheet。

### 2. 環境依賴
```bash
pip install streamlit plotly pandas openpyxl numpy
```

## 🎨 UI 設計規範 (Luxury Cream Gold)
- **色調**：奶油象牙金 (`#f3efe8`), 香檳金 (`#c9a66b`), 炭黑文字 (`#1f1f1f`)。
- **效果**：大面積使用 `backdrop-filter: blur(10px)` 玻璃擬態效果與 20px 圓角面板。
- **交互**：所有圖表需適配 `Cream Gold Lux Edition` 主題，禁止使用系統預設配色。

## 🛠 維護與故障排除
- **數據不更新？**：點擊側邊欄「🔄 強制刷新取數」按鈕。
- **圖表報錯？**：檢查 Excel 列名是否有空格。系統已內置 `pick_col` 工具進行模糊匹配，但建議保持表頭清潔。
- **新增渠道？**：修改 `map_channel` 函數中的關鍵詞匹配列表。
